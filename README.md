# 🧬 刺激と反応・排出 フラッシュカード

中学3年 理科（生物）「刺激と反応」「排出」の1学期中間テスト対策用、全60問の一問一答フラッシュカードです。

**公開URL**：[https://y-redhat.github.io/-/](https://y-redhat.github.io/-/)

---

## 📚 収録内容

| カテゴリ | 問題数 | 内容 |
|----------|--------|------|
| 🔵 基礎 | 25問 | 用語の定義・器官の名称など基礎知識 |
| 🟢 仕組み | 15問 | 反射の経路・暗順応・筋肉の連動などプロセス理解 |
| 🟣 応用 | 20問 | 恒常性・進化的意義・テクノロジー応用など高校基礎レベルを含む考察問題 |

---

## 🎯 使い方

1. カードを**タップ**すると解答と解説が表示されます
2. もう一度タップすると問題面に戻ります
3. **◀ ▶ ボタン**で前後の問題に移動
4. **🔀 シャッフル**で出題順をランダムに入れ替え
5. 上部のプログレスバーで現在の進捗が確認できます
